## 🌐 Live Site
Deployed on Vercel: [prostar-cnc.vercel.app](https://prostar-cnc.vercel.app)

## 📁 Structure
```
prostar-site/
├── index.html          # الصفحة الرئيسية
├── about.html          # من نحن
├── machines.html       # الماكينات
├── services.html       # الخدمات (تشمل الصيانة)
├── contact.html        # تواصل معنا
├── quote.html          # طلب عرض سعر
├── careers.html        # الوظائف
├── blog.html           # المدونة
├── article.html        # صفحة المقال
├── faq.html            # الأسئلة الشائعة
├── images/             # صور الماكينات والشعار
│   └── logo-new.png
├── vercel.json         # Vercel config
└── package.json
```

## 🚀 Deploy
1. Push to GitHub
2. Connect repo to Vercel
3. Auto-deploys on every push
