# إعداد Web3Forms لاستقبال طلبات العملاء

## الخطوة 1: إنشاء حساب (دقيقتين)
1. ادخل على https://web3forms.com
2. اكتب إيميلك واضغط "Create Access Key"
3. افتح إيميلك وانسخ الـ Access Key

## الخطوة 2: أضف الـ Key في الملفات
ابحث عن `YOUR_ACCESS_KEY_HERE` في هذه الملفات وبدلها بالـ Key:
- quote.html (نموذج طلب عرض سعر)
- contact.html (نموذج تواصل معنا)

## الخطوة 3: ارفع على GitHub
ارفع الملفات المحدثة وVercel هيعمل deploy تلقائي.

## النتيجة
كل طلب عرض سعر أو رسالة تواصل هتوصلك على إيميلك مباشرة!
المجاني يسمح بـ 250 رسالة شهرياً.
